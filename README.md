# Calculator React App
:pushpin: With React library created Calculator using Vite tool.

> Visit : https://calculator-app-vikas.vercel.app/

* Library : `React`
* Package Manager : `npm`
* Used Tool : `Vite`  
* Deployed On : `Vercel`

> To Install :
```
npm install
```
> To Use :
```
npm run dev
```
